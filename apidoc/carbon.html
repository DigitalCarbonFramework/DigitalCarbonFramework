carbon.compute_footprints.adcalls_cost(framework, nb_ad_calls: int, creative_type: Literal['video', 'display']) → AdcallCost[source]¶

Return the kgco2 cost of a number of ad calls. We approximate the number of SSPs connected to be 10 on average for each prebid auction. The Framework defines a number of paths activated at each bid: 1 for direct auction, 100 for video ad, 350 for display ad. Thus, each adcall activates 10 paths for a video ad, 35 for a display ad.

Parameters:

  • framework (Framework) – Framework object

  • nb_ad_calls (int) – number of ad calls

  • creative_type (Literal[&quot;video&quot;, &quot;display&quot;]) – Type of the creative

Returns:

a AdcallCost object containing the Co2 cost of a number of ad calls.

Return type:

AdcallCost
